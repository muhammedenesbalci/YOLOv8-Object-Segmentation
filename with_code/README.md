## YOLOv8 object segmentation with Code

**Contents**

 - Automatic segmentation for an image.
 - Customized segmentation for an image.
 - Automatic segmentation for a video.
 - Customized segmentation for a video.

**Requirements**

Pip install the ultralytics package including all requirements.
```
pip install ultralytics  
```
